암호화 또는 서명 첫 번째 HGS 서버 인증서에 대 한 PFX 파일을 제공 하지 않은 경우이 서버에 공개 키만 복제 됩니다.
개인 키를 로컬 인증서 저장소로 또는 HSM 백업 키의 경우 개인 키가 들어 있는 PFX 파일을 가져와서 설치 해야 합니다. 키 저장소 공급자를 구성 하 고 HSM 제조업체 별로 인증서를 사용 하 여 연결 지침입니다.
