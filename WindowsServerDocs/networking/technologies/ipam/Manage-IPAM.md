# <a name="manage-ipam"></a>IPAM 관리

>적용 대상: Windows Server (반기 채널), Windows Server 2016

이 가이드에서는 관리 및 Windows Server 2016에서 관리 IPAM (IP 주소) 기능에 대 한 문제 해결 정보를 제공 합니다.
