# <a name="branchcache-network-shell-and-windows-powershell-commands"></a>네트워크 BranchCache 셸 및 Windows PowerShell 명령

>적용 대상: Windows Server (세미콜론 연간 채널) Windows Server 2016

Windows Server 2016 구성 하 고 BranchCache BranchCache에 대 한 Windows PowerShell 또는 네트워크 셸 (Netsh) 명령을 사용 하 여 관리할 수 있습니다.  
  
나중에 버전의 Windows에서는 Microsoft 제거 될 수 있습니다 netsh 기능 BranchCache에 대 한 합니다. Microsoft는 현재 netsh 구성 하 고 관리 BranchCache 및 기타 네트워킹 기술을 사용 하는 경우 해당 사용자 전환이를 Windows PowerShell 권장 합니다.  
  
Windows PowerShell 및 netsh 명령 참조 다음 위치에 있습니다. 두 명령 참조 Windows Server 2016 이전 운영 체제에 게시 된 되지만 이러한 참조는이 운영 체제에 대 한 정확한입니다.  
  
-   [Netsh 명령을 BranchCache Windows Server 2008 r 2에서에 대 한](https://technet.microsoft.com/library/dd979561(v=ws.10))  
  
-   [Windows powershell에서 Cmdlet BranchCache](https://technet.microsoft.com/library/hh848392.aspx) Windows Server 2012에 대 한 합니다.  
  
> [!TIP]  
> 프롬프트에서 Windows PowerShell BranchCache에 대 한 Windows PowerShell 명령 목록을 보려면 입력 `Get-Command -Module BranchCache` 프롬프트를 Windows PowerShell 및 ENTER 키를 누릅니다.
