# <a name="step-10-install-and-configure-2-edge1"></a>10 설치 단계 및 2 EDGE1 구성

>적용 대상: Windows Server (반기 채널), Windows Server 2016

2-EDGE1 구성 중 구성 됩니다.  
  
- 2-EDGE1에 운영 체제를 설치 합니다. 2-EDGE1에 Windows Server 2016, Windows Server 2012 R2 또는 Windows Server 2012 설치 합니다.  
  
- TCP/IP 속성을 구성합니다. 두 네트워크 인터페이스에 정적 주소를 사용 하 여 2 EDGE1를 구성 합니다.  
  
- 서브넷 간 라우팅을 구성 합니다. Corpnet 및 2 Corpnet 서브넷 간의 통신을 사용 하려면 라우팅을 구성 해야 합니다.  
  
- 2-EDGE1 CORP2 도메인에 조인 합니다. 2-EDGE1 corp2.corp.contoso.com 도메인에 조인 합니다.  
  
- 2-EDGE1에서 인증서를 가져옵니다. 인증서는 클라이언트가 HTTPS를 통해 연결 하는 경우 IP-HTTPS 수신기 인증할 및 DirectAccess 클라이언트 및 원격 액세스 서버 간의 IPsec 연결에 필요 합니다.  
  
- CORP\User1에 대 한 액세스를 제공 합니다. CORP\User1 사용자는 원격 액세스 관리자입니다. 변경 하려면 2-EDGE1 EDGE1에서이 사용자를 사용 하려면 사용자에 게 액세스를 부여 해야 합니다.  
  
- 2-EDGE1 원격 액세스 역할을 설치 합니다. 멀티 사이트 배포를 사용 하려면 2-EDGE1에서 원격 액세스 역할을 설치 해야 합니다.  
  
2-EDGE1 두 네트워크 어댑터가 설치 되어 있어야 합니다.  
  
## <a name="installOS"></a>2-EDGE1에서 운영 체제를 설치 합니다.  
  
1.  Windows Server 2016, Windows Server 2012 R2 또는 Windows Server 2012의 설치를 시작 합니다.  
  
2.  Windows Server 2016, Windows Server 2012 R2 또는 Windows Server 2012 (전체 설치) 및 로컬 관리자 계정의 강력한 암호를 지정 하 여 설치를 완료 하려면 지침을 따릅니다. 로컬 관리자 계정을 사용하여 로그온합니다.  
  
3.  2-EDGE1 인터넷에 연결 하는 네트워크에 연결 하 고 Windows Server 2016, Windows Server 2012 R2 또는 Windows Server 2012에 대 한 최신 업데이트를 설치 하 고 인터넷에서 연결을 해제 하려면 Windows Update를 실행 합니다.  
  
4.  하나의 네트워크 어댑터 2 Corpnet 서브넷 및 시뮬레이션된 된 인터넷에 연결 합니다.  
  
## <a name="tcpip"></a>TCP/IP 속성 구성  
  
1.  서버 관리자 콘솔에서 클릭 **로컬 서버**를 선택한 다음는 **속성** 영역에서 다음을 **유선 이더넷 연결**, 링크를 클릭 합니다.  
  
2.  **네트워크 연결**2 Corpnet 서브넷에 연결 되어 있는 네트워크 연결을 마우스 오른쪽 단추로 클릭, 클릭 **이름 바꾸기**, 유형 **2 Corpnet**, 한 다음 ENTER를 누릅니다.  
  
3.  마우스 오른쪽 단추로 클릭 **2 Corpnet**를 클릭 하 고 **속성**합니다.  
  
4.  **Internet Protocol Version 4(TCP/IPv4)** 를 클릭하고 **속성**을 클릭합니다.  
  
5.  **다음 IP 주소 사용**을 클릭합니다. **IP 주소**, 형식 **10.2.0.20**의 **서브넷 마스크**, 형식 **255.255.255.0**합니다.  
  
6.  **다음 DNS 서버 주소 사용**을 클릭합니다. **기본 설정 DNS 서버**, 형식 **10.2.0.1**, 및 **대체 DNS 서버**, 형식 **10.0.0.1**합니다.  
  
7.  **고급**을 클릭하고 **DNS** 탭을 클릭합니다.  
  
8.  **이 연결에 대 한 DNS 접미사**, 형식 **corp2.corp.contoso.com**를 클릭 하 고 **확인** 두 번입니다.  
  
9. **인터넷 프로토콜 버전 6(TCP/IPv6)**, **속성**을 차례로 클릭합니다.  
  
10. 클릭 **다음 IPv6 주소를 사용 하 여**입니다. **IPv6 주소**, 형식 **2001:db8:2::20**의 **서브넷 접두사 길이**, 형식 **64**합니다. 클릭 **다음 DNS 서버 주소를 사용 하 여**, 및 **기본 설정 DNS 서버**, 유형 **2001:db8:2::1**의 **대체 DNS 서버**, 형식 **2001:db8:1::1**합니다.  
  
11. **고급**을 클릭하고 **DNS** 탭을 클릭합니다.  
  
12. **이 연결에 대 한 DNS 접미사**, 형식 **corp2.corp.contoso.com**를 클릭 하 고 **확인** 두 번입니다.  
  
13. 에 **2 Corpnet 속성** 대화 상자, 클릭 **닫기**합니다.  
  
14. 에 **네트워크 연결** 창에서 인터넷 서브넷에 연결 되어 있는 네트워크 연결을 마우스 오른쪽 단추로 클릭, 클릭 **이름 바꾸기**, 형식 **인터넷**, enter .  
  
15. **인터넷**을 마우스 오른쪽 단추로 클릭한 다음 **속성**을 클릭합니다.  
  
16. **Internet Protocol Version 4(TCP/IPv4)** 를 클릭하고 **속성**을 클릭합니다.  
  
17. **다음 IP 주소 사용**을 클릭합니다. **IP 주소**, 형식 **131.107.0.20**합니다. **서브넷 마스크**에 **255.255.255.0**을 입력합니다.  
  
18. **고급**을 클릭합니다. **IP 설정** 탭의 **IP 주소** 영역에서 **추가**를 클릭합니다. 에 **TCP/IP 주소** 대화 상자의 **IP 주소** 형식 **131.107.0.21**의 **서브넷 마스크** 형식 **255.255.255.0** 를 클릭 하 고 **추가**합니다.  
  
19. **DNS** 탭을 클릭합니다.  
  
20. **이 연결에 대 한 DNS 접미사**, 형식 **isp.example.com**, 클릭 **확인** 을 두 번 클릭 하 고 **닫기**합니다.  
  
21. **네트워크 연결** 창을 닫습니다.  
  
## <a name="routing"></a>서브넷 간 라우팅 구성  
  
1.  에 **시작** 화면에서 입력**cmd.exe**, 한 다음 ENTER를 누릅니다.  
  
2.  명령 프롬프트 창에서 다음 명령을 입력 합니다. 각 명령을 입력 한 후 enter 키를 누릅니다.  
  
    ```  
    netsh interface IPv4 add route 10.0.0.0/24 2-Corpnet 10.2.0.254  
    netsh interface IPv6 add route 2001:db8:1::/64 2-Corpnet 2001:db8:2::fe  
    ```  
  
3.  2-EDGE1과 DC1 간의 네트워크 통신을 확인 하려면 입력 **ping dc1.corp.contoso.com**합니다.  
  
4.  확인에서 4 개의 응답 10.0.0.1, IPv4 주소 또는 IPv6 주소 2001:db8:1::1에서에서.  
  
5.  명령 프롬프트 창을 닫습니다.  
  
## <a name="Join"></a>2-EDGE1 CORP2 도메인에 가입  
  
1.  서버 관리자 콘솔에서의 **로컬 서버**를 **속성** 영역에서 다음을 **컴퓨터 이름**, 링크를 클릭 합니다.  
  
2.  **시스템 속성** 대화 상자의 **컴퓨터 이름** 탭에서 **변경**을 클릭합니다.  
  
3.  에 **컴퓨터 이름/도메인 변경** 대화 상자의 **컴퓨터 이름**, 유형 **2 EDGE1**합니다. **소속**, 클릭 **도메인**, 유형 **corp2.corp.contoso.com**를 클릭 하 고 **확인**합니다.  
  
4.  사용자 이름 및 암호에 대 한 메시지가 표시 되 면 입력 **관리자** 암호 및 클릭 **확인**합니다.  
  
5.  Corp2.corp.contoso.com 도메인 시작 대화 상자, 표시 되 면 클릭 **확인**합니다.  
  
6.  컴퓨터를 다시 시작해야 한다는 메시지가 표시되면 **확인**을 클릭합니다.  
  
7.  **시스템 속성** 대화 상자에서 **닫기**를 클릭합니다.  
  
8.  컴퓨터를 다시 시작할지 묻는 메시지가 표시되면 **지금 다시 시작**을 클릭합니다.  
  
9. 컴퓨터를 다시 시작한 후를 클릭 **사용자 전환**를 클릭 하 고 **기타 사용자** 및 관리자 계정으로 CORP2 도메인에 로그온 합니다.  
  
## <a name="certs"></a>2-EDGE1에서 인증서 가져오기  
  
1.  에 **시작** 화면에서 입력**mmc.exe**, 한 다음 ENTER를 누릅니다.  
  
2.  MMC 콘솔에서에 **파일** 메뉴 클릭 **스냅인 추가/제거**합니다.  
  
3.  에 **추가 / 제거 스냅인** 대화 상자를 클릭 **인증서**, 클릭 **추가**, 클릭 **컴퓨터 계정**, 클릭 **다음**, 클릭 **로컬 컴퓨터**, 클릭 **마침**, 클릭 하 고 **확인**합니다.  
  
4.  인증서 스냅인의 콘솔 트리에서 엽니다 **인증서 (로컬 컴퓨터) \Personal**합니다.  
  
5.  마우스 오른쪽 단추로 클릭 **개인**를 가리키고 **모든 작업**를 클릭 하 고 **새 인증서 요청**합니다.  
  
6.  **다음** 을 두 번 클릭합니다.  
  
7.  에 **인증서 요청** 페이지에서를 **Client-server Authentication** 및 **웹 서버** 확인란을 선택한 다음 클릭 **자세한 정보 이 인증서를 등록 하는 데 필요한**합니다.  
  
8.  에 **인증서 속성** 대화 상자의 **주체** 탭의 **주체 이름** 영역에서 **형식**선택,  **일반 이름**합니다.  
  
9. **값**, 형식 **2 edge1.contoso.com**를 클릭 하 고 **추가**합니다.  
  
10. 에 **대체 이름** 영역에서 **형식**, 선택, **DNS**합니다.  
  
11. **값**를 입력 **2 edge1.contoso.com**를 클릭 하 고 **추가**합니다.  
  
12. 에 **일반** 탭의 **친숙 한 이름**, 유형 **IP-HTTPS 인증서**합니다.  
  
13. 클릭 **확인**, 클릭 **등록**, 를 클릭 하 고 **마침**합니다.  
  
14. 인증서 스냅인의 세부 정보 창에서 서버 인증 용도로, 이름 2 edge1.contoso.com를 사용 하 여 새 인증서가 등록 하 고 이름 2 edge1.corp2.corp.contoso.com를 사용 하 여 새 인증서를 사용 하 여 등록 된 확인 클라이언트 인증 및 서버 인증의 의도 한 목적으로 합니다.  
  
15. 콘솔 창을 닫습니다. 설정을 저장 하는 메시지가 누릅니다 **No**합니다.  
  
## <a name="Access"></a>CORP\User1에 액세스할 수  
  
1.  에 **시작** 화면에서 입력**compmgmt.msc**, 한 다음 ENTER를 누릅니다.  
  
2.  왼쪽된 창에서 **로컬 사용자 및 그룹**합니다.  
  
3.  두 번 클릭 **그룹**, 를 두 번 클릭 하 고 **관리자**합니다.  
  
4.  에 **관리자 속성** 대화 상자를 클릭 **추가**, 및는 **사용자, 컴퓨터, 서비스 계정 또는 그룹 선택** 대화 상자를 클릭 **위치**합니다.  
  
5.  에 **위치** 대화 상자의 합니다 **위치** 트리를 클릭 **corp.contoso.com**를 클릭 하 고 **확인**.  
  
6.  에 **선택할 개체 이름을 입력 하십시오** 형식 **User1**를 클릭 하 고 **확인**합니다.  
  
7.  에 **관리자 속성** 대화 상자를 클릭 하 여 **확인**합니다.  
  
8.  컴퓨터 관리 창을 닫습니다.  
  
## <a name="InstallDA"></a>2-EDGE1 원격 액세스 역할 설치  
  
1.  서버 관리자 콘솔에서에 **대시보드**, 클릭 **역할 및 기능 추가**합니다.  
  
2.  **다음** 을 세 번 클릭하여 서버 역할 선택 화면으로 이동합니다.  
  
3.  **서버 역할 선택** 대화 상자에서 **원격 액세스**를 선택하고 **기능 추가**를 클릭한 후 **다음**을 클릭합니다.  
  
4.  **다음**을 5번 클릭합니다.  
  
5.  **설치 선택 확인** 대화 상자에서 **설치**를 클릭합니다.  
  
6.  **설치 진행률** 대화 상자에서 설치가 완료되었는지 확인하고 **닫기**를 클릭합니다.
