## <a name="TCP"></a>RSA에서 TCP/IP를 구성 합니다.  
  
1.  초기 구성 작업을 클릭 **네트워킹 구성**합니다.  
  
2.  **네트워크 연결**를 마우스 오른쪽 단추로 클릭 **로컬 영역 연결**를 클릭 하 고 **속성**합니다.  
  
3.  **Internet Protocol Version 4(TCP/IPv4)** 를 클릭하고 **속성**을 클릭합니다.  
  
4.  **다음 IP 주소 사용**을 클릭합니다. **IP 주소**에 **10.0.0.5**를 입력합니다. **서브넷 마스크**에 **255.255.255.0**을 입력합니다. **기본 게이트웨이**, 형식 **10.0.0.2**합니다. 클릭 **다음 DNS 서버 주소를 사용 하 여**의 **기본 설정 DNS 서버**, 유형 **10.0.0.1**합니다.  
  
5.  **고급**을 클릭하고 **DNS** 탭을 클릭합니다.  
  
6.  **이 연결에 대 한 DNS 접미사**, 형식 **corp.contoso.com**를 클릭 하 고 **확인** 두 번입니다.  
  
7.  에 **로컬 영역 연결 속성** 대화 상자, 클릭 **닫기**합니다.  
  
8.  **네트워크 연결** 창을 닫습니다.
