# <a name="install-updates"></a>업데이트 설치
사용 가능한 경우 업데이트를 설치 하는 것이 좋습니다. 업데이트를 설치 하려면 인터넷에 연결 합니다.  

1.  **시작** 화면에서 **제어판**을 엽니다.  
  
2.  제어판에서 입력 **업데이트**를 클릭 하 고 **업데이트 확인**합니다.  
  
3.  Windows 업데이트 웹 사이트를 컴퓨터에 필요한 모든 업데이트를 나열, 업데이트를 설치 합니다.
