# Windows에서 원격 데스크톱을 사용 하 여 시작

>적용 대상: Windows 10, Windows 8.1, Windows Server 2019, Windows Server 2016, Windows Server 2012 R2

Windows 앱 및 다른 Windows 장치에서 원격 데스크톱을 사용 하려면 Windows에 대 한 원격 데스크톱 클라이언트를 사용할 수 있습니다.

시작 하려면 다음 정보를 사용 합니다. 질문이 있는 경우 [FAQ](remote-desktop-client-faq.md) 를 확인 해야 합니다.

> [!NOTE]
> - Windows 클라이언트에 대 한 새 릴리스 용어나? 체크 아웃 [Windows에서 원격 데스크톱에 대 한 새로운 기능?](windows-whatsnew.md)
> - 모든 버전의 Windows 10에서 클라이언트를 실행할 수 있습니다.

## RD 클라이언트를 가져오고 사용을 시작합니다

Windows 10 디바이스에 원격 데스크톱을 사용 하 여 시작 하려면 다음이 단계를 따르세요.

1. [Microsoft Store](https://www.microsoft.com/store/p/microsoft-remote-desktop/9wzdncrfj3ps)에서 원격 데스크톱 클라이언트를 다운로드 합니다. 
2. [원격 연결을 허용 하도록 PC를 설정](remote-desktop-allow-access.md)합니다.
3. 원격 데스크톱 연결 또는 원격 리소스를 추가 합니다. 연결을 사용 하 여 RemoteApp 프로그램, 세션 기반 데스크톱 또는 관리자가 게시 하는 가상 데스크톱을 사용 하 여 Windows PC 및 원격 리소스에 직접 연결 하려면 
4. 항목을 고정 하 여 원격 데스크톱에 빠르게 액세스할 수 있습니다.

### 원격 데스크톱 연결을 추가 합니다.

원격 데스크톱 연결을 만들려면:

1. 연결 센터에서 **+ 추가**, 탭 하 고 **바탕 화면**을 탭 합니다.
2. 연결 하려는 컴퓨터에 대 한 다음 정보를 입력 합니다.
  - **PC 이름** -컴퓨터의 이름입니다. 이 Windows 컴퓨터 이름, 인터넷 도메인 이름으로 또는 IP 주소일 수 있습니다. 또한 PC 이름 (예를 들어 **MyDesktop:3389** 또는 **10.0.0.1:3389**) 포트 정보를 추가할 수 있습니다.
  - **사용자 계정** – 원격 PC에 액세스 하는 데 사용자 계정입니다. 탭 **+** 새 계정을 추가 하거나 기존 계정을 선택 합니다. 다음과 같은 형식을 사용 하 여 사용자 이름에 대 한: *사용자 _*, *domain\user_name*또는 *user_name@domain.com*합니다. **매 번 확인**을 선택 하 여 연결 되어 있는 동안 사용자 이름 및 암호를 표시할지 여부를 지정할 수 있습니다.
3. 설정할 수도 있습니다 추가 옵션 탭 하면 **더 많은 표시**:
  - **표시 이름** -에 연결 하는 PC에 대 한 기억 하기 쉬운 이름을입니다. 모든 문자열을 사용할 수 있지만 이름을 지정 하지 않으면 PC 이름이 표시 됩니다.
  - **그룹** -쉽게 나중에 연결을 찾을 수 있도록 그룹을 지정 합니다. 탭 하 여 새 그룹을 추가할 수 있습니다 **+** 목록에서 하나를 선택 합니다.
  - **게이트웨이** -가상 데스크톱, RemoteApp 프로그램 및 세션 기반 데스크톱에서 회사 내부 네트워크에 연결 하는 데 사용할 수 있는 원격 데스크톱 게이트웨이. 시스템 관리자에서 게이트웨이 대 한 정보를 가져옵니다.
  - **관리자 세션에 연결** -Windows server 관리 콘솔 세션에 연결 하려면이 옵션을 사용 합니다.
  - **스왑 마우스 단추** -마우스 오른쪽 단추에 대 한 마우스 왼쪽된 단추 기능을 교체 하려면이 옵션을 사용 합니다. (이 경우 특히 유용 왼손 사용자에 대 한 원격 PC가 구성 되었지만 오른손 마우스를 사용 합니다.)
  - **내 원격 세션 해상도를 설정:** – 세션에 사용 하려는 해상도 선택 합니다. **내가 사용 하도록 선택** 클라이언트의 크기에 따라 해상도 설정 됩니다.
  - **디스플레이의 크기를 변경:** – 세션에 대 한 정적 높은 해상도 선택할 때는 옵션이 표시 가독성을 높이기 위해 더 큰 화면에서 항목을 만들 수 있습니다. 참고:이 때만 적용 됩니다 이상 또는 Windows 8.1 연결 합니다.
  - **해상도에서 크기를 조정 하 여 원격 세션이 업데이트** – 사용 하는 경우, 클라이언트는 동적으로 업데이트 클라이언트의 크기에 따라 세션 해상도 합니다. 참고:이 때만 적용 됩니다 이상 또는 Windows 8.1 연결 합니다.
  - **클립보드** – 사용 하면 텍스트와 이미지 원격 PC에서 복사할 수 있습니다.
  - **오디오 재생** – 원격 세션 동안 오디오를 사용 하 여 장치를 선택 합니다. 원격 PC 로컬 장치에서 하거나 전혀 없는 소리를 재생 선택할 수 있습니다.
  - **오디오 녹음/녹화** – 사용 하면 원격 PC에서 응용 프로그램을 사용 하 여 로컬 마이크를 사용할 수 있습니다.
4. **저장**을 누릅니다.

이러한 설정을 편집할 필요가? 바탕 화면 이름 옆에 있는 오버플로 메뉴 (**...**) 누르고 누른 다음 **편집**합니다.

연결을 삭제 하려면? 마찬가지로 오버플로 메뉴 (**...**)를 누르고 **제거**합니다.

### 원격 리소스를 추가 합니다.
원격 리소스는 RemoteApp 프로그램, 세션 기반의 데스크톱 및 원격 데스크톱 서비스를 사용 하 여 사용자의 관리자가 게시 하는 가상 데스크톱.

에 원격 리소스를 추가 합니다.

1. 연결 센터 화면에서 **+ 추가**, 탭 하 고 **원격 리소스**를 누릅니다. 
2. 귀하의 관리자에서 제공한 **피드 URL** 을 입력 하 고 **찾기 피드를**누릅니다.
3. 대화 상자가 나타나면 피드 구독을 사용 하 여 자격 증명을 제공 합니다.

원격 리소스 연결 가운데에 표시 됩니다.


원격 리소스를 삭제 합니다.

1. 연결 센터에서 원격 리소스 옆의 오버플로 메뉴 (**...**)를 누릅니다.
2. **제거**를 누릅니다.

### 시작 메뉴에 저장 된 바탕 화면에 고정

시작 메뉴에 대 한 연결에 고정 하려면 데스크톱, 이름 옆에 있는 오버플로 메뉴 (**...**) 누르고 **시작 화면에 고정**합니다.

이제 탭 하 여 시작 메뉴에서 직접 원격 데스크톱 연결을 시작할 수 있습니다.

## 내부 자산에 액세스할 수 있는 RD 게이트웨이에 연결

원격 데스크톱 게이트웨이 (RD 게이트웨이)를 사용 하면 어디에서 나 회사 네트워크에서 원격 컴퓨터에 연결에서 인터넷 합니다. 만들 수 있으며 원격 데스크톱 클라이언트를 사용 하 여 게이트웨이 관리.

새로운 게이트웨이를 설정 하려면:

1. 연결 중앙에서 **설정**을 누릅니다.
2. 게이트웨이, 옆에 있는 탭 **+** 새로운 게이트웨이 추가 합니다. 참고: 새 연결을 추가할 때 게이트웨이 추가할 수도 있습니다.
3. 다음 정보를 입력 합니다.
  - **서버 이름** -게이트웨이 사용 하려는 컴퓨터의 이름입니다. 이 Windows 컴퓨터 이름, 인터넷 도메인 이름으로 또는 IP 주소일 수 있습니다. 서버 이름으로 포트 정보를 추가할 수도 있습니다 (예: **RDGateway:443** 또는 **10.0.0.1:443**).
  - **사용자 계정** 연결 하는 원격 데스크톱 게이트웨이 함께 사용 하려면 사용자 계정을 추가 하거나 선택 합니다. **데스크톱 사용자 계정을 사용 하 여** 원격 데스크톱 연결에 사용 되는 것과 동일한 자격 증명을 사용 하 여 선택할 수 있습니다.
4. **저장**을 누릅니다.  

## 글로벌 앱 설정

**설정**탭 하 여 클라이언트에서 다음 전역 설정을 설정할 수 있습니다.

항목을 관리
- **사용자 계정** 을 추가 하면, 편집 하 고 클라이언트에 저장 된 사용자 계정을 삭제 합니다. 이 변경 된 후에 계정에 대 한 암호를 업데이트 하는 것이 좋습니다.
- **게이트웨이** 를 추가 하면, 편집 하 고 클라이언트에 저장 된 게이트웨이 서버를 삭제 합니다.
- **그룹** 에 추가 하면, 편집 하 고 클라이언트에 저장 된 그룹을 삭제 합니다. 이러한 그룹 연결 쉽게 수 있습니다.

세션 설정
- **전체 화면에서 연결 시작** -사용 하면 연결 시작 언제 든 지 클라이언트가 현재 모니터의 전체 화면을 사용 합니다.
- **새 창에서 각 연결 시작** -사용 하면 각 연결에서에서 실행 되는지 별도 창, 다른 모니터에 배치 하 고 작업 표시줄을 사용 하 여 간에 전환할 수 있습니다.
- **앱 크기를 조정할 때:** -클라이언트 창 크기를 조정 하는 경우를 제어할 수 있습니다. 기본값은 **가로 세로 비율을 유지 하 여 콘텐츠를 확장**합니다.
- **사용 하 여 키보드 명령을 사용 하 여:** - *승리* 또는 *ALT + TAB* 같은 키보드 명령을 사용 되는 위치를 지정 하겠습니다. 기본값은 전체 scren에 연결 되 면 세션에만 보낼 합니다.
- **화면 시간을 초과 하지** -시간을 초과 세션 활성 상태일 때 화면을 유지할 수 있습니다. 오랜 시간 동안 연결 상호 작용이 필요 하지 않습니다 때 유용 합니다.

앱 설정
- **데스크톱 미리 보기 표시** -에 연결 하기 전에 연결 중심의 데스크톱의 미리 보기를 볼 수 있습니다. 기본적으로이 **on**으로 설정 됩니다.
- **원격 데스크톱을 향상 시키기 위해** -익명 데이터를 Microsoft에 보냅니다. 클라이언트를 개선 하기 위해이 데이터를 사용 합니다. 이 익명, 개인 데이터를 처리 하는 방법에 대 한 자세한 내용은 [Microsoft 개인정보취급방침](https://privacy.microsoft.com/en-us/privacystatement)를 참조 하세요. **기본적으로이 설정은 켜져 있습니다.**

### 사용자 계정 관리

데스크톱 또는 원격 리소스에 연결할 때 다시에서 선택 하도록 사용자 계정을 저장할 수 있습니다. 또한 사용자 데이터를 저장 하 여 데스크톱에 연결할 때가 아니라 클라이언트 자체의 사용자 계정을 정의할 수 있습니다.

새 사용자 계정을 만들려면:

1. 연결 중앙에서 **설정**을 누릅니다.
2. 사용자 계정 옆에 있는 탭 **+** 새 사용자 계정을 추가 합니다.
3. 다음 정보를 입력 합니다.
   - **사용자 이름** -원격 연결에 사용 하기 위해 저장 하는 사용자의 이름입니다. 다음 형식 중 하나로 사용자 이름을 입력할 수 있습니다: 사용자 _, domain\user_name, 또는 user_name@domain.com 합니다.
   - **암호** -지정한 사용자에 대 한 암호입니다. 이 비워둘 수 연결 되어 있는 동안 암호를 입력 해야 합니다.
4. **저장**을 누릅니다.

사용자 계정을 삭제 합니다.

1. 연결 중앙에서 **설정**을 누릅니다.
2. 사용자 계정 목록에서 삭제 하려면 계정을 선택 합니다.
3. 사용자 계정 옆에 있는 편집 아이콘을 탭 합니다.
4. 사용자 계정을 삭제 하려면 맨 아래에 **이 계정을 제거** 를 누릅니다.
5. 사용자 계정 편집 하 고 **저장**탭 수도 있습니다.

## 원격 데스크톱 세션 이동
원격 데스크톱 연결을 시작할 때 사용할 수 있는 세션 이동 하는 데 사용할 수 있는 도구가 있습니다.

### 원격 데스크톱 연결을 시작 합니다.

1. 세션을 시작에 대 한 원격 데스크톱 연결을 누릅니다.
2. 연결에 대 한 자격 증명을 저장 하지 않은 경우 **사용자 이름** 및 **암호**를 제공 하 라는 메시지가 표시 됩니다.
3. 원격 데스크톱에 대 한 인증서를 확인 하는 메시지가 정보를 검토 하 고이 PC **연결**탭 하기 전에 신뢰할 수 있는지 확인 합니다. 항상이 인증서를 수락 하도록 **다시이 인증서에 대 한 요청 하지 않도록** 선택할 수 있습니다.

### 연결 표시줄

연결 표시줄은 추가 탐색 컨트롤에 대 한 액세스 제공 합니다. 기본적으로 연결 표시줄 화면 맨 위에 있는 가운데에 배치 됩니다. 탭 하 고 왼쪽 또는 오른쪽으로 이동 하는 막대를 끕니다.

- **이동 컨트롤** -이동 컨트롤 화면을을 확대 하 고 이동할 수 있습니다. 참고 이동 컨트롤은 에서만 터치 사용 디바이스 및 직접 터치 모드를 사용 하 여 사용할 수 있습니다.
   - 활성화 / 이동 컨트롤을 사용 하지 않도록 설정: 화면을 확대/축소 및 이동 컨트롤을 표시 하려면 연결 표시줄에 이동 아이콘을 탭 합니다. 연결 표시줄 숨기기를 기존 픽셀 화면 돌아갑니다에 팬 아이콘을 탭 합니다.
   - 탭 이동 컨트롤을 사용 하 고 이동 컨트롤 화면 이동 하려는 방향으로 끌어 합니다.
   - 두 번 탭 이동 컨트롤 이동 하 고 화면에서 컨트롤을 이동 제어할 수 있습니다.
- **추가 옵션** -세션 선택 영역 바 및 명령 표시줄 (아래 참조)를 표시 하려면 탭 추가 옵션 아이콘입니다.
- **키보드** -탭 키보드 아이콘을 표시 하거나 숨기는 화상 키보드 합니다. 이동 컨트롤이 키보드 표시 될 때 자동으로 표시 됩니다.

### 명령 모음

화면의 오른쪽에 있는 명령 모음을 표시 하도록 연결 모음의 **...** 를 탭 합니다.

- **홈** 홈 단추를 사용 하 여 명령 모음에서 연결 중심으로 돌아갑니다.
  - 또는 동일한 작업에 대해 뒤로 단추를 사용할 수 있습니다.
  - 활성 사용자 세션을 중단 합니다.
  - 이렇게 하면 추가 연결을 시작할 수 있습니다.
- **연결 끊기** -연결 끊기 단추를 사용 하 여 여 연결을 종료 합니다.
  - 원격 PC에 세션이 종료 되지으로 앱 활성화 유지 됩니다.
- **전체 화면** 전체 화면 모드로 들어가거나 합니다.
- **마우스 / 터치** -(직접 터치 및 마우스 포인터)는 마우스 모드 간에 전환할 수 있습니다.

### 원격 세션에서 사용 하 여 직접 터치 제스처 및 마우스 모드

두 마우스 모드는 세션 상호 작용할 수 있습니다.
- **직접 터치**: 모든 터치 연락처를 원격으로 해석 하도록 세션에 전달 합니다.
  - 터치 스크린을 사용 하 여 Windows 사용 하 여 동일한 방식으로 사용 합니다.
- **마우스 포인터**: 로컬 터치 화면의 세션에서 마우스 포인터를 이동 하는 큰 터치 패드 허용으로 변환 합니다.
  - Winodws 사용 하 여 터치 패드와 동일한 방식으로 사용 합니다.

> [!NOTE]
> Windows 8와의 상호 작용 또는 최신 직접 터치 모드에서 기본 터치 제스처 지원 됩니다. 

| 마우스 모드    | 마우스 조작      | 제스처                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 직접 터치  | 마우스 왼쪽 단추로 클릭           | 1 손가락으로 탭 하기                                                          |
| 직접 터치  | 마우스 오른쪽 단추로 클릭          | 1 손가락으로 탭 하기 및 누르고 있기                                                 |
| 마우스 포인터 | 마우스 왼쪽 단추로 클릭           | 1 손가락으로 탭 하기                                                          |
| 마우스 포인터 | 왼쪽 클릭 후 끌기  | 두 번 탭 1 손가락으로 누른 다음 끕니다.                               |
| 마우스 포인터 | 마우스 오른쪽 단추로 클릭          | 2 손가락으로 탭 하기                                                          |
| 마우스 포인터 | 오른쪽 클릭 후 끌기 | 2 손가락으로 두 번 탭 하기 및 고정, 다음 끕니다.                               |
| 마우스 포인터 | 마우스 휠          | 2 손가락으로 탭 하 고, 계속 누르고 위 또는 아래로 끌기                           |
| 마우스 포인터 | Zoom                 | 확대 또는 축소 손가락 떨어져 이동 손가락 모으기 및 2 손가락을 사용 합니다. |
