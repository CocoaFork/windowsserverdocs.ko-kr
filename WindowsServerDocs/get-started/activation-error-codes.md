# 정품 인증 오류 코드의 문제 해결

**홈 사용자** 이 문서는 IT 전문가 지원 상담원이 사용할 것입니다. Windows 정품 인증 오류 메시지에 대 한 자세한 정보를 찾고 있는 경우 다음 Microsoft 웹 사이트에 이동 합니다.

[Windows 정품 인증 오류에 대한 도움말 보기](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

하나 이상의 Windows 기반 컴퓨터에서 볼륨 정품 인증을 수행 하는 여러 MAK 정품 인증 키 () 또는 (KMS (키 관리 서비스)를 사용 하려고 할 때 특정 오류 코드를 포함 하는 오류 메시지가 나타납니다. 이 문서에서는 이러한 오류를 해결 하는 방법을 설명 합니다.
