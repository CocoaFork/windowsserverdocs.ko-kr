# <a name="bitsadmin-sethttpmethod"></a>bitsadmin sethttpmethod

HTTP 동사를 사용 하 여 설정 합니다.

## <a name="syntax"></a>구문

```
bitsadmin /GetHttpMethod <Job> <HTTPMethod>
```

## <a name="parameters"></a>매개 변수

|매개 변수|설명|
|---------|-----------|
|작업|작업의 표시 이름 또는 GUID|
|HTTPMethod|HTTP 동사를 사용 하 여|
