# <a name="bitsadmin-peers-and-list"></a>bitsadmin 피어 및 목록



모든 피어를 나열합니다.

## <a name="syntax"></a>구문

```
bitsadmin /Peers /List 
```

## <a name="BKMK_examples"></a>예제

다음 예제에서는 모든 피어를 나열합니다.
```
C:\>bitsadmin /Peers /List
```
