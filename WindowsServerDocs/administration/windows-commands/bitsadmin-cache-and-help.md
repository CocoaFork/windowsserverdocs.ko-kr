# <a name="bitsadmin-cache-and-help"></a>bitsadmin 캐시 및 도움말



**캐시** 스위치에 대 한 명령줄 사용법을 표시 합니다.

## <a name="syntax"></a>구문

```
Bitsadmin /Cache /Help 
```

## <a name="BKMK_examples"></a>예와

다음 명령은 **캐시** 스위치에 대 한 명령줄 도움말을 표시 합니다.
```
Bitsadmin /Cache /Help
```
