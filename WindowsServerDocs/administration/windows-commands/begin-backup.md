# <a name="begin-backup"></a>백업 시작



전체 백업 세션을 시작 합니다.

## <a name="syntax"></a>구문

```
begin backup
```

## <a name="remarks"></a>설명

-   이 명령은 기본 복사 백업 설정을 무시합니다.
