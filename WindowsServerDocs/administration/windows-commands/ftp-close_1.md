# <a name="ftp-close_1"></a>ftp: close_1

>적용 대상: Windows Server (반기 채널), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

원격 서버와 ftp 세션을 종료 하 고 **ftp >** 프롬프트로 유지 합니다.   
## <a name="syntax"></a>구문  
```  
close  
```  
### <a name="parameters"></a>매개 변수  
없음

