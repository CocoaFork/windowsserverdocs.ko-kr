# <a name="nslookup-set-debug"></a>nslookup set debug



디버깅 모드를 설정 하거나 해제 합니다.

## <a name="syntax"></a>구문

```
set [no]debug
```

## <a name="parameters"></a>매개 변수

|  매개 변수  |                         설명                          |
|-------------|--------------------------------------------------------------|
| **nodebug** | 디버깅 모드를 해제합니다. 기본 구문은 **nodebug**합니다. |
|  **debug**  |                   디버깅 모드를 설정합니다.                   |
|  \*\*{도움말  |                            ?}\*\*                            |

## <a name="remarks"></a>설명

-   디버깅 모드를 켜면, 서버 및 응답을 보내는 방법에 대 한 자세한 내용은 인쇄 됩니다.
