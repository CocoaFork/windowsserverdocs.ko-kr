# <a name="bitsadmin-help"></a>bitsadmin help



명령줄 사용법을 표시합니다.

## <a name="syntax"></a>구문

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>예제

다음 예제에서는 명령줄 도움말을 검색합니다.
```
C:\>bitsadmin /help
```
