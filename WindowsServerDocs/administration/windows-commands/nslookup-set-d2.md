# <a name="nslookup-set-d2"></a>nslookup set d2



철저 한 디버깅 모드를 설정 하거나 해제 합니다. 모든 패킷이 모든 필드를 표시 합니다.

## <a name="syntax"></a>구문

```
set [no]d2
```

## <a name="parameters"></a>매개 변수

| 매개 변수 |                             설명                              |
|-----------|----------------------------------------------------------------------|
| **nod2**  | 철저 한 디버깅 모드를 해제합니다. 기본 구문은 **nod2**합니다. |
|  **d2**   |                 철저 한 디버깅 모드를 설정합니다.                  |
|   {도움말   |                                  ?}                                  |
