# <a name="automount"></a>자동 탑재



자동 탑재 기능을 사용 하지 않도록 설정 하거나 사용 합니다. (기본값)를 사용 하도록 설정 하면 시스템에 추가 되 고 다음 볼륨에 드라이브 문자를 할당 하는 경우 Windows가 자동으로 새 기본 또는 동적 볼륨에 대 한 파일 시스템을 탑재 합니다.
