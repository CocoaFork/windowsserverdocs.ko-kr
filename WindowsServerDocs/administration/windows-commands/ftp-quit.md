# <a name="ftp-quit"></a>ftp: 종료

>적용 대상: Windows Server (반기 채널), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

원격 컴퓨터와 ftp 세션을 종료 하 고 **ftp** 를 종료 하 여 운영 체제 명령 프롬프트를 표시 합니다.   
## <a name="syntax"></a>구문  
```  
quit  
```  
### <a name="parameters"></a>매개 변수  
없음

```  
quit  
```
