# <a name="bitsadmin-peers-and-help"></a>bitsadmin 피어 및 도움말



**피어** 스위치에 대 한 명령줄 사용법을 표시 합니다.

## <a name="syntax"></a>구문

```
Bitsadmin /Peers /Help 
```

## <a name="BKMK_examples"></a>예와

다음 명령은 **피어** 스위치에 대 한 명령줄 도움말을 표시 합니다.
```
Bitsadmin /Peers /Help
```
