# <a name="vol"></a>vol



있는 경우 디스크의 볼륨 레이블과 일련 번호를 표시 합니다.  매개 변수 없이 사용 하는 경우 **vol** 현재 드라이브에 대 한 정보를 표시 합니다.

## <a name="syntax"></a>구문

```
vol [<Drive>:]
```

## <a name="parameters"></a>매개 변수

|매개 변수|설명|
|---------|-----------|
|\<Drive >:|볼륨 레이블 및 일련 번호를 표시 하려는 디스크를 포함 하는 드라이브를 지정 합니다.|
|/?|명령 프롬프트에 도움말을 표시합니다.|
