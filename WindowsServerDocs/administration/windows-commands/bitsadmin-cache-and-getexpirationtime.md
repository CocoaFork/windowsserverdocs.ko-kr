# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin 캐시 및 getexpirationtime



캐시 만료 시간을 검색합니다.

## <a name="syntax"></a>구문

```
bitsadmin /Cache /GetExpirationTime 
```

## <a name="BKMK_examples"></a>예와

다음 캐시 만료 시간을 검색합니다.
```
C:\>bitsadmin /Cache /GetExpirationTime
```
