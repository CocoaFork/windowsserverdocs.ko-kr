# <a name="bitsadmin-peers-and-clear"></a>bitsadmin 피어 및 지우기



피어 목록을 지웁니다.

## <a name="syntax"></a>구문

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>예제

다음 예제에서는 피어 목록을 지웁니다.
```
C:\>bitsadmin /Peers /Clear
```
